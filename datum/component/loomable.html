/tg/ Station 13 - Modules - Types — Var Details - Proc Details

loomable
/datum/component/loomable

Component that makes items turn into other items when you use them on a loom (or any other thing really if you change the var)

Vars

loom_timeHow long it takes to loom the parent
process_completion_verbWhat verb best fits the action of processing whatever the item is, for example "spun [thing]"
required_amountHow much of parent do we need to loom, will be ignored if parent isnt a stack
resulting_itemWhat will spawn when the parent is loomed
target_needs_anchoringIf target_thing needs to be anchored
target_thingWhat thing we look for triggering the loom process (usually a loom)

Procs

loom_meIf a do_after of the specified loom_time passes, will create a new one of resulting_item and either delete the parent, or .use the required amount if its a stack
on_examineAdds an examine blurb to the description of any item that can be loomed
try_and_loom_meChecks if the thing we clicked on can be used as a loom, and if we can actually loom the parent at present (an example being does the stack have enough in it (if its a stack))
